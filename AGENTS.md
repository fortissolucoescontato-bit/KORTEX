# Kortex — Índice de Skills para Agentes

Ao trabalhar neste projeto, carregue a(s) skill(s) relevante(s) ANTES de escrever qualquer código.

## Como Usar

1. Verifique a coluna "Gatilho" para encontrar skills que correspondam à sua tarefa atual.
2. Carregue a skill lendo o arquivo `SKILL.md` no caminho listado.
3. Siga TODOS os padrões e regras da skill carregada.
4. Múltiplas skills podem ser aplicadas simultaneamente.

## Skills

| Skill | Gatilho | Caminho |
|-------|---------|------|
| `kortex-issue-creation` | Ao criar uma issue no GitHub, reportar um bug ou solicitar uma funcionalidade. | [`skills/issue-creation/SKILL.md`](skills/issue-creation/SKILL.md) |
| `kortex-branch-pr` | Ao criar um pull request, abrir um PR ou preparar alterações para revisão. | [`skills/branch-pr/SKILL.md`](skills/branch-pr/SKILL.md) |
