# Padrões de Engenharia Kortex

Este é um projeto **open-source** de elite desenvolvido pela **Nexo-Fortis**. Nossa missão é tornar a programação com IA acessível a todos, e sua contribuição é fundamental para alcançarmos esse objetivo.

Seja você um iniciante querendo aprender ou um sênior querendo ajudar, todos são bem-vindos para contribuir mantendo nossos padrões de excelência.

## 📐 Princípios de Desenvolvimento

1. **Análise de Cenário**: Antes de qualquer alteração, deve-se realizar uma análise 360 do impacto.
2. **Persona de Elite**: O sistema opera como um Analista de Elite — brasileiro, polido e estratégico. Seu *modus operandi* inclui:
    - **Análise 360°**: Avaliar todos os cenários antes de qualquer ação.
    - **Mentoria**: Explicar o "porquê" das ações para empoderar o usuário.
    - **Antecipação**: Sugerir o próximo passo lógico proativamente.
    - **Resiliência**: Sempre apresentar uma solução junto com qualquer reporte de erro.
3. **Código de Alta Performance**: Priorize eficiência, legibilidade e segurança.
4. **Padrões Go**: Siga as melhores práticas da comunidade Go 1.25.
5. **Persistência Atômica**: Todas as alterações de estado devem ser feitas via `state.Manager`.

## 🛠️ Processo de Desenvolvimento

- Use Conventional Commits.
- Mantenha a cobertura de testes em dia.
- Documente novas funcionalidades no `openspec/`.
